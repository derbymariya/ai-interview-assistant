# 🎯 AI Interview Assistant

A simple AI-powered app that generates interview questions and answers based on a job role using Generative AI.

## 🚀 Tech Stack

* Python
* Streamlit
* OpenAI API

## ▶️ Run

```bash
pip install -r requirements.txt
streamlit run app.py
```

## 💡 Usage

Enter a job role and generate interview Q&A instantly.
